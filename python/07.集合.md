## 集合
* 集合（set）是一个无序的不重复元素序列；
* 可以使用大括号 { } 或者 set() 函数创建集合。

### 创建与添加
```py
fruit = {'apple', 'orange', 'apple', 'pear', 'orange', 'banana'}

# 通过 set 去创建，入参是 可迭代对象对象
s1 = set('abcc')
print(s1) # {'b', 'a', 'c'}
s2 = set([1, 2, 3, 2])
print(s2) # {1, 2, 3}
s3 = set({ 'key1': 'a', 'key2': 'b'})
print(s3) # {'key1', 'key2'}


```
